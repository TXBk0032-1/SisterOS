## 使用说明

### 启动系统
```bash
cd sisters_flower_system
python run.py
```

### 主要功能操作

#### 销售管理
1. 在"销售管理"标签页
2. 输入会员手机号 (可选)
3. 输入商品名称和数量
4. 点击"确认销售"

#### 会员管理
1. 在"会员管理"标签页
2. 点击"新增会员"添加新会员
3. 点击"会员充值"进行余额充值
4. 点击"查询会员"查看会员信息

#### 库存管理
1. 在"库存管理"标签页
2. 点击"新增商品"添加新商品
3. 点击"编辑商品"修改商品信息
4. 点击"删除商品"移除商品

#### 数据统计
1. 在"数据统计"标签页
2. 查看今日销售数据
3. 设置销售目标
4. 查看热门商品排行

#### 系统管理
1. 在"系统管理"标签页
2. 点击"创建备份"备份数据
3. 点击"系统信息"查看系统状态
4. 点击"退出系统"关闭程序
