# 🌸 姐妹花销售系统 4.0 快速启动指南

## 📋 系统要求
- Windows 10/11 (推荐) 或 Linux/macOS
- Python 3.8+ 
- 至少 2GB RAM，100MB 磁盘空间

## 🚀 方法一：一键启动（推荐）
```bash
# 1. 进入系统目录
cd code/sisters_flower_system

# 2. 运行快速启动
python enhanced_sales_system.py
```

## 🔑 默认登录信息
- **用户名**: admin
- **密码**: admin

## 🆘 常见问题

### Q: 忘记管理员密码怎么办？
A: 重新运行 `python enhanced_sales_system.py --reset-password`

### Q: 如何切换主题？
A: 点击侧边栏底部的主题切换按钮

### Q: 数据库文件在哪里？
A: 位于 `sisters_flowers.db`，建议定期备份

### Q: 系统支持多语言吗？
A: 当前支持中文和英文，可扩展其他语言
